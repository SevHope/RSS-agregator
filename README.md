<h3>System requirements</h3>
<p>Node.js version - v18.14.1</p>
<h3>install dependencies</h3>
<p>make install</p>
<h3>build the webpack</h3>
<p>`make build`</p>
<h3>run the webpack</h3>
<p>make start</p>
